# 籃球戰術板

純 HTML + CSS + JavaScript 實作的籃球戰術板 Web 應用，無任何框架或外部依賴，可在桌機與行動裝置上直接使用。

## 功能

- **球場切換**：半場 / 全場即時切換
- **球員棋子**：進攻（橘色）、防守（藍色）各最多 5 名，自動編號 1–5
- **籃球**：可獨立放置與移動的籃球棋子
- **拖曳操作**：從底部工具列拖曳新增棋子至場上；場上棋子可自由拖曳重新定位
- **刪除**：將棋子拖曳至垃圾桶區域即可移除
- **復原**：支援最多 40 步的復原（Undo）操作
- **清空**：一鍵清除場上所有棋子
- **行動端支援**：完整支援 Touch events，響應式版面配置

## 使用方式

直接用瀏覽器開啟 `index.html`，無需安裝任何套件或啟動伺服器。

```
open index.html
```

## 操作說明

| 動作 | 桌機 | 行動端 |
|------|------|--------|
| 新增棋子 | 從工具列拖曳至場上 | 從工具列觸碰並拖曳至場上 |
| 移動棋子 | 直接拖曳場上棋子 | 觸碰並拖曳場上棋子 |
| 刪除棋子 | 拖曳至右下角垃圾桶 | 觸碰並拖曳至右下角垃圾桶 |
| 復原 | 點擊 ↩ 按鈕 | 點擊 ↩ 按鈕 |
| 清空全部 | 點擊 ✕ 按鈕 | 點擊 ✕ 按鈕 |
| 切換球場 | 點擊「半場 / 全場」 | 點擊「半場 / 全場」 |

> 切換半場 / 全場時，場上所有棋子將會清除。

## 技術架構

- **單一檔案**：所有 HTML、CSS、JavaScript 集中於 `index.html`
- **繪圖引擎**：Canvas 2D API，含 DPR（devicePixelRatio）高解析度支援
- **棋子系統**：以 Canvas 物件實作，非 DOM 元素
- **事件處理**：使用 Pointer Events API 統一處理滑鼠與觸控輸入
- **球場規格**：依 NBA 標準尺寸（半場 50×47 ft、全場 94×50 ft）等比繪製
